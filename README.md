# ReactNativePractice
`React`를 통해 웹뿐만 아니라 안드로이드 등의 모바일 앱도 만들 수 있는 `React Native`를 연습하였습니다.

# React Native를 배우고자 하는 이유
- `React`의 철학 중 하나인 `한 번 배워서 어디서나 사용하기`를 느끼기 위해.
- 웹뿐만 아니라 모바일 앱 개발에서도 `Java`나 `Kotlin`을 대신함으로써 생산성을 더욱 향상시키기 위해.
